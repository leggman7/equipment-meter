# Mining Equipment Hour Meter Tracker

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript

## Project Goal
Build a website that allows users to view and update hour meter readings on mining equipment.

## Pages
- **Main Page (`/`)** — Displays a list of all mining equipment. Each item shows the equipment name and its current hour meter reading. Users can update the hour meter reading directly from this page.
- **About Page (`/about`)** — Explains that the purpose of this website is to track and update hour meter readings on mining equipment.

## Style
- Clean and minimal UI.
- Consistent spacing, typography, and color usage across all pages.
- Layout should be fully responsive across desktop, tablet, and mobile.

## Rules
- Do not add any comments in the code.
- Do not add unnecessary libraries or dependencies.
- Keep file and folder structure clean and organized.
- All pages must include navigation to move between Main and About.
- Data must persist after a page refresh.
